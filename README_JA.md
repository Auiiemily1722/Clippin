# PinClip - クリップボード履歴管理ツール

[简体中文](README.md) | [English](README_EN.md) | [繁體中文](README_ZH-TW.md)

## 概要

グラスモーフィズム（Glassmorphism）デザインを採用したモダンなクリップボード履歴管理ツール。

## 技術スタック

- **フロントエンド**: HTML + CSS + JavaScript（単一ファイル）
- **バックエンド**: Electron
- **データベース**: SQLite (sql.js)

## 機能

| 機能 | 説明 |
|------|------|
| ウィンドウ操作 | ドラッグ可能、リサイズ可能（8方向） |
| 最小化 | macOSスタイルのアニメーション |
| 閉じる | スケールフェードアウトアニメーション |
| 日付フィルター | ドロップダウン：すべて/今日/昨日/一昨日/それ以前 |
| 検索 | リアルタイムフィルタリング |
| カードグループ化 | 時間別にグループ表示 |
| ピン留め | ピンアイコンでカードを固定（FLIPアニメーション） |
| コピー | コピーアイコンまたはカードをクリック、チェックマークアニメーション |
| 削除 | カードが左にスライドして消える |
| プレビュー | 長いテキストをホバーで展開（グラス効果） |
| 一括操作 | 複数選択モード、一括ピン留め/削除 |

### デザイン詳細

- ウィンドウ角丸: 10px
- グラス効果: `backdrop-filter: blur(40px) saturate(180%)`
- アニメーション曲線: `cubic-bezier(0.25, 0.1, 0.25, 1)`
- 配色: ダークグラデーション背景 + 白いグラスカード
- 信号機ボタン: 赤で閉じる、黄で最小化、緑でピン留め（macOSスタイル）

## インストール

```bash
npm install
npm start
```

## ビルド

```bash
npm run build:installer  # インストーラー
npm run build:portable   # ポータブル版
npm run build            # すべて
```

## 特徴

- **FLIPアニメーション**: すべてのリスト操作にスムーズな位置アニメーション
- **グラスモーフィズム**: backdrop-filterでスモークドグラス効果
- **多言語対応**: EN / ZH-CN / ZH-TW / JA
- **ダークモード**: システム自動追従または手動切り替え
- **グローバルホットキー**: Ctrl+Shift+V で素早く起動
