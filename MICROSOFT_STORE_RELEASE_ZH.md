# Clippin 微软商店上架准备说明

## 产品定位

Clippin 是一款 Windows 本地剪贴板历史管理工具。它面向经常复制、整理、复用文本内容的用户，核心价值是让用户更快找回复制过的文字，并通过搜索、置顶、批量管理和快捷键提升日常办公效率。

## 核心卖点

- 本地优先：剪贴板历史保存在用户设备上，不依赖账号、云同步或服务器。
- 隐私清晰：应用不上传剪贴板内容，不集成广告、分析或第三方数据服务。
- 高频效率：支持搜索、日期筛选、置顶、批量删除、批量置顶和长文本预览。
- 桌面体验：支持系统托盘、全局快捷键、窗口动画和多语言界面。
- 轻量精致：相比系统自带剪贴板历史，Clippin 更适合长期整理和复用文本片段。

## 推荐收费策略

- 中国大陆首发价：人民币 8 元。
- 其他主要市场首发价：2.49 美元或当地等值价格。
- 收费模式：一次性买断。
- 暂不建议订阅：当前版本是本地工具，尚无云同步、账号体系或持续服务成本，订阅理由不够强。
- 暂不建议高价：剪贴板工具赛道成熟，且项目曾以 MIT 许可证开源，用户对价格会更敏感。

## 上架前必须完成

- 在 Partner Center 中确认应用最终名称为 `Clippin`，并确保该名称可用。
- 将 Partner Center 给出的最终 `Publisher`、`Package/Identity/Name`、`Publisher display name` 回填到 `package.json` 的 `build.appx` 配置中。
- 重新运行 `npm run build:store`，生成品牌名为 `Clippin` 的新 AppX 包。
- 将 `privacy-policy.html` 发布到公开可访问的网址，例如 GitHub Pages、官网或静态托管服务。
- 重新整理商店截图，建议使用统一语言、统一窗口尺寸和统一视觉风格。
- 准备支持邮箱，建议继续使用 `Clippin.studio@gmail.com`。
- 在商店描述中明确说明：剪贴板内容仅保存在本地设备，不会上传服务器。
- 使用 AppX 包在本机完成安装、启动、托盘、快捷键、剪贴板监听和卸载回归测试。

## 建议商店文案

### 简短描述

Clippin 是一款本地优先的 Windows 剪贴板历史管理工具，帮助你快速找回、搜索、置顶和复用复制过的文本。

### 长描述

Clippin 为经常处理文字内容的用户提供更高效的剪贴板历史体验。它会在本地记录你复制过的文本，让你可以通过搜索、日期筛选、置顶和批量管理快速找回需要的内容。

所有剪贴板数据都保存在你的设备上。Clippin 不需要账号，不上传剪贴板内容，也不集成广告、分析工具或第三方数据服务。它适合写作、编程、客服、运营、翻译和日常办公等需要频繁复制粘贴的场景。

主要功能包括：

- 自动记录文本剪贴板历史。
- 快速搜索复制过的内容。
- 按日期筛选历史记录。
- 置顶常用片段。
- 批量删除和批量置顶。
- 长文本悬停预览。
- 系统托盘驻留。
- 全局快捷键快速唤起。
- 多语言界面。
- 本地 SQLite 数据存储。
